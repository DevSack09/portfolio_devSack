Proyectos

Sistema Integral de Postulación y Control

Sistema web desarrollado para la gestión del proceso de registro, validación y control documental. La plataforma automatiza tareas clave tanto para participantes como para administradores, ofreciendo una experiencia segura, eficiente y accesible desde cualquier dispositivo.


HTML • CSS • JavaScript • Bootstrap • MySQL • PHP

PaqueTesiga – Sistema de Monitoreo en Tiempo Real

Aplicación móvil y sistema web, diseñada para el monitoreo en tiempo real del traslado de paquetes electorales. La app permite escanear códigos QR y registrar eventos geolocalizados durante el recorrido de los paquetes, mientras que el sistema web ofrece la visualización en tiempo real.


HTML • CSS • JavaScript • Bootstrap • MySQL • PHP • Kotlin • Firebase

Sor Juana Inés de la Cruz

Sistema web diseñada para la gestión integral de registros académicos de aspirantes y alumnos. Permite consultar materias, registrar inscripciones y reinscripciones a cursos, así como acceder a calificaciones.


XML • CSS • JavaScript • Bulma • PostgreSQL • Java

Gestión de Servicios (SGS)

Aplicación web que facilita la creación, seguimiento y administración de tickets de soporte y servicios tecnológicos. Incluye módulos para usuarios, áreas, departamentos, reportes gráficos, calendario de actividades y notificaciones automáticas. Diseñado para ser intuitivo, responsivo y seguro, garantizando una gestión ágil y eficiente.


HTML • CSS • JavaScript • Bootstrap • MySQL • PHP

Captura de Actas y Cómputos para el Proceso Electoral Local

Sistema web para la captura precisa y oportuna de los resultados electorales en el estado de Hidalgo. La plataforma permite registrar actas para gestionar cómputos electorales y ofrecer un módulo avanzado de consulta mediante gráficos e indicadores visuales además de mostrar el seguimiento del proceso en tiempo real.


HTML • CSS • JavaScript • Bootstrap • MySQL • PHP
